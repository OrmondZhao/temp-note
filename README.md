# 临时素材板

一个在浏览器中随时打开、用于临时收集和查看文本与图片素材的轻量网页应用。

## 快速启动

### 方式一：快捷方式（推荐）

桌面上已有 **Temp Note** 快捷方式，双击即可一键启动服务器并在浏览器中打开。

### 方式二：命令行启动

```bash
cd D:\Project\temp-note
npm start
```

然后在浏览器中访问 http://localhost:3173

## 功能说明

- **文本记录**：输入标题、正文、标签，保存后可随时查看
- **截图粘贴**：直接 `Ctrl+V` 粘贴剪贴板截图，或拖拽图片文件到编辑区
- **多附件**：同一卡片可保存多张截图
- **自动保存**：内容自动保存在浏览器本地 IndexedDB，刷新不丢失
- **搜索筛选**：按关键词搜索、按类型（文本/图片）筛选、按置顶/收藏筛选
- **置顶收藏**：重要内容可置顶或收藏，方便快速查找
- **导入导出**：支持单条素材的 JSON 导出与导入
- **明暗主题**：点击"暗黑模式"按钮切换

## 数据存储

- **IndexedDB**：素材内容（文本、图片）保存在浏览器本地
- **localStorage**：界面状态（搜索词、筛选条件、主题偏好）
- **exports/ 文件夹**：导出的 JSON 备份文件

## 快捷键（浏览器原生）

| 按键 | 功能 |
|------|------|
| `Ctrl + Enter` | 保存当前内容 |
| `Esc` | 关闭图片预览 / 取消新建 |

## 注意事项

- 图片数据量较大时，浏览器本地存储空间有限，请定期导出重要内容备份
- 清除浏览器缓存会导致本地数据丢失，重要数据请及时导出
- 仅在本地使用时数据保存在本机，不支持跨设备同步
