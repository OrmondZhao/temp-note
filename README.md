# 临时素材板

一个在浏览器中随时打开、用于临时收集和查看文本与图片素材的轻量网页应用。

## 访问地址

**在线访问（无需任何配置）：** https://note.kzhao29.win

## 功能说明

* **富文本编辑**：基于 Quill 编辑器，支持加粗、斜体、标题、列表、代码块、链接等格式化编辑
* **文本记录**：输入标题、正文、标签，输入时自动保存
* **图文混排**：编辑区支持文本和图片混合编辑，图片以 Base64 编码内联存储
* **截图粘贴**：直接 `Ctrl+V` 粘贴剪贴板截图，或拖拽图片文件到编辑区
* **多附件**：同一卡片可保存多张截图
* **自动保存**：内容输入时自动保存在浏览器本地 IndexedDB，刷新不丢失
* **搜索筛选**：按关键词搜索、按类型（文本/图片）筛选、按置顶/收藏筛选
* **置顶收藏**：重要内容可置顶或收藏，方便快速查找
* **导入导出**：支持单条素材的 JSON 导出与导入；导出时自动保存卡片内容，导入后样式格式完全一致；复制内容为纯文本（无格式）
* **明暗主题**：点击"暗黑模式"按钮切换，包含富文本编辑器的暗黑样式适配

## 数据存储

* **IndexedDB**：素材内容（文本、图片）保存在浏览器本地
* **localStorage**：界面状态（搜索词、筛选条件、主题偏好）
* **导出文件**：导出操作将 JSON 文件下载到本地（浏览器下载目录）

## 本地开发

如果你想修改代码并本地测试：

```bash
# 克隆项目
git clone <仓库地址>
cd temp-note

# 安装依赖
npm install

# 启动本地开发服务器
npm start
```

然后在浏览器访问 http://localhost:3173。本地开发模式下，导出的 JSON 文件会保存在 `exports/` 文件夹中。

## 注意事项

* 图片数据量较大时，浏览器本地存储空间有限，请定期导出重要内容备份
* 清除浏览器缓存会导致本地数据丢失，重要数据请及时导出
* 数据仅保存在当前浏览器设备，不支持跨设备同步
